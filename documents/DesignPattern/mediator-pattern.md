# Mediator Pattern

미디에이터 패턴(Mediator Pattern), 우리말로는 중재자 패턴이라고 한다. 클래스 간의 복잡한 상호작용을 캡슐화 하여 한 클래스에 위임하여 처리하는 패턴이다. 

객체간의 통신을 중재자(Mediator) 라는 객체에게 맡기고 서로 직접 통신하지 않도록 하여 객체간의 결합도를 낮추기 위한 패턴이다
